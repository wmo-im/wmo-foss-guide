== AI Tool Use in FOSS

=== Overview of Issues

A Scientific Python blog post, https://discuss.scientific-python.org/t/blog-community-considerations-around-ai-contributions/2234[Community Considerations Around AI Contributions], provides a good introduction to concerns and possible benefits of AI tool use.

=== Examples of FOSS Project AI Tool Use Policies

- GDAL https://gdal.org/en/stable/community/ai_tool_policy.html[LLM tool policy]
- matplotlib https://matplotlib.org/devdocs/devel/contribute.html#generative-ai[Use of Generative AI guidelines]
- NumPy - https://numpy.org/devdocs/dev/ai_policy.html[AI Policy]
- OpenJDK https://openjdk.org/legal/ai[Interim Policy on Generative AI]
- Python https://devguide.python.org/getting-started/generative-ai/[Generative AI guidelines]
- QGIS - https://github.com/qgis/QGIS-Enhancement-Proposals/blob/master/qep-408-ai-tool-policy.md[AI tool use policy]
- Zarr-python https://zarr.readthedocs.io/en/stable/contributing/#ai-assisted-contributions[AI-assisted contributions guidelines]
